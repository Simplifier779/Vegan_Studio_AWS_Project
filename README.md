# The_Vegan_Studio_AWS_Project

# 🥑 The Vegan Recipe Contest Web App

A community-driven web application built for **The Vegan Studio** to host a public recipe contest. This app allows users to submit their favorite vegan recipes, with the best recipe being selected and featured across The Vegan Studio’s platforms.
---

## 📖 Project Description

This project supports the **“Favourite Vegan Recipe Contest”** initiated by The Vegan Studio’s marketing and culinary teams. The goal is to boost brand identity, drive engagement, and grow the vegan community through a fun and rewarding challenge.

👤 Users can:
- Submit one vegan recipe with ingredients and preparation steps
- View the contest rules and submission deadlines

👩‍🍳 Admins (chef team) can:
- View and review all submissions
- Select and highlight the winning recipe
---

## 🧠 Features

- 📝 Recipe submission form with validations
- 🔒 User authentication (optional via Cognito or Firebase)
- 📋 Admin dashboard to view and manage entries
- 🏆 Display section for the winning recipe
- 📢 Community & engagement section
